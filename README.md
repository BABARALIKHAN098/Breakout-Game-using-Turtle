# Breakout-Game-using-Turtle
## Features - **Paddle Movement**: Control the paddle to bounce the ball. - **Ball Physics**: Ball bounces off walls, paddle, and bricks. - **Score &amp; Lives**: Track your score and remaining lives (starts with 3). - **Bricks**: Break all bricks to win. - **Win/Loss**: Game over if you run out of lives; Win if you clear all bricks.
