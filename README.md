# Breakout Game

A classic Breakout arcade game implemented using Python's `turtle` module.

## Features
- **Paddle Movement**: Control the paddle to bounce the ball.
- **Ball Physics**: Ball bounces off walls, paddle, and bricks.
- **Score & Lives**: Track your score and remaining lives (starts with 3).
- **Bricks**: Break all bricks to win.
- **Win/Loss**: Game over if you run out of lives; Win if you clear all bricks.

## Prerequisites
- Python 3.x (The `turtle` module is included in the standard library).

## How to Run
1.  Navigate to the game directory.
2.  Run the game script:
    ```bash
    python main.py
    ```

## Controls
- **Left Arrow**: Move Paddle Left
- **Right Arrow**: Move Paddle Right
